# x86.h Changes

## Overview
Added support for loading the Global Descriptor Table and removed unused code.

## Detailed Changes
- **lgdt**: New inline function to load a GDT pointer:
  ```c
  static inline void
  lgdt(struct segdesc *p, int size) {
    unsigned char pd[6];

    pd[0] = size;
    pd[1] = size >> 8;
    pd[2] = (unsigned long)p;
    pd[3] = (unsigned long)p >> 8;
    pd[4] = (unsigned long)p >> 16;
    pd[5] = (unsigned long)p >> 24;
    asm volatile("lgdt (%0)" : : "r" (pd));
  }
  ```
- **Removed `readeflags`**: The function reading EFLAGS was deleted.

## Impact
The new `lgdt` helper is used by `vm.c` to install the GDT set up during boot. It enables the segmentation changes introduced in this diff. Removing `readeflags` cleans up unused code.
