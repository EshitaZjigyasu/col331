# x86.h Changes

## Overview
The `x86.h` file was updated to add the `lgdt` function for loading the Global Descriptor Table and remove the `readeflags` function.

## Detailed Changes
- **lgdt function**: Added a new inline function to load the GDT:
  - Takes a pointer to `struct segdesc` and size.
  - Constructs a pseudo-descriptor array.
  - Uses `asm volatile("lgdt (%0)")` to load the GDT.
- **readeflags function**: Removed the function that reads the EFLAGS register.

## Impact
The `lgdt` function is crucial for setting up segmentation by loading the GDT into the GDTR register. Removing `readeflags` simplifies the header, as it's no longer needed in the current implementation.
