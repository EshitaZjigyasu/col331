# .gdbinit.tmpl Changes

## Overview
A small update was made to the GDB initialization template used for xv6 kernel debugging.

## Detail
- The line `layout split` was commented out by prefixing it with `#`.

## Reasoning
This change disables the automatic split-window layout when GDB starts. Users can still manually enable the layout if desired, but the default now avoids a split view which may be preferable in certain terminal configurations.
